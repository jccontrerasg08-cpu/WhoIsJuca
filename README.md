# Juan Carlos Contreras — Portfolio

Static site (GitHub Pages ready). Open `index.html`.

## Structure
```
index.html               Home / portfolio (entry point)
profile.html             Same as index (alias)
carta-presentacion.html  Cover letter (ES/EN, 3 types, editable fields)
assets/                  Photos
cv/                      Résumés (open in browser, Print → PDF)
  cv-en.html · cv-es.html            Business Analyst (EN/ES)
  cv-en-data.html · cv-es-data.html  Data Analyst (EN/ES)
lib/                     Shared code (cv-ats.css, doc-page.js, tweaks-panel.jsx)
downloads/               Ready files (.docx, .pdf)
```

## GitHub Pages
Settings → Pages → deploy from branch → root. Site serves `index.html`.
